# STM32电子桌宠 (Electronic Desktop Pet)

![项目封面图](这里放一张你做的实物最漂亮的照片)

## 项目简介
这是一个基于 **STM32F103C8T6** 微控制器开发的电子桌宠装置。该项目实现了与电子桌宠的养成互动、环境监测及数据更新可视化功能，包含完整的硬件PCB设计与嵌入式软件开发。

> 2025学年 ESE 启蒙题目作品 / 个人独立完成

## 功能特性 (Features)
* [1]**核心交互**：OLED屏幕显示宠物动画，支持“快乐、饥饿、疲倦”三维状态管理。
* [2]**多模式时钟**：支持普通模式（30s周期）与极速演示模式（1s周期）切换。
* [3]**环境感知**：集成AHT20传感器，实时监测并显示环境数据]。
* [4]**数据持久化**：采用EEPROM模块实现数据掉电存储，每分钟自动保存宠物状态。
* [5]**数据可视化**：在OLED屏幕上绘制最近1分钟的属性波动曲线，曲线会向左流畅滑动更新 。
* [6]**远程控制**：支持蓝牙无线控制与状态回传。

## 硬件架构 (Hardware)
* **MCU**: STM32F103C8T6
* **Display**: 0.96寸 OLED (I2C)
* **Sensor**: AHT20 温湿度传感器
* **Input**: 独立按键 x 4 (含消抖处理)
* **Output**: 无源蜂鸣器 (可调节音频及音量)
* **PCB**: 自主设计的双层板 (嘉立创EDA专业版)

## 软件架构
* 使用 STM32CubeMX 生成初始化代码。
* 采用结构体以及枚举语法实现数据封装，使代码清晰易懂。
* 实现了简单的任务调度与状态机管理。
